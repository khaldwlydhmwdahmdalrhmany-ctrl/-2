# مجلد الخطوط الخاصة بالموقع (Custom Fonts Directory)

هذا المجلد مخصص لحفظ ملفات الخطوط الخاصة بموقع نثال لتنقية المياه (مثل ملفات `.woff` أو `.woff2` أو `.ttf`).

## كيفية الاستخدام:
1. قم برفع ملفات الخطوط إلى هذا المجلد.
2. قم بربط الخطوط في ملف الاستايل الرئيسي `src/index.css` باستخدام `@font-face`.
